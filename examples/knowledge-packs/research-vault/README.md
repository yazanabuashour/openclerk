# Research Vault Knowledge Pack

This pack shows citation-bearing search over small research notes and one
source-linked synthesis page.

Contents:

- `vault/sources/` has synthetic research notes.
- `vault/synthesis/retrieval-notes.md` summarizes those notes with source refs.
- `commands.md` shows search and synthesis lifecycle inspection.

Expected outcome: search returns cited source snippets, and projection
inspection shows synthesis freshness from canonical Markdown sources.

# Handwritten Inbox Knowledge Pack

This pack simulates shorthand or OCR-like notes using plain Markdown. The notes
are candidate evidence, not canonical truth.

Contents:

- `vault/sources/` has reviewed source context.
- `inbox/shorthand-note.md` has unclear shorthand awaiting review.
- `commands.md` shows candidate planning.

Expected outcome: unclear shorthand becomes a needs-review candidate with open
questions. Durable writes still require approval.
